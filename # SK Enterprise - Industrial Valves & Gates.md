# SK Enterprise - Industrial Valves & Gates

A modern web application for SK Enterprise, specializing in manufacturing industrial valves and gates, built with React and Tailwind CSS.

## 🚀 Features

- Modern, responsive design with Framer Motion animations
- Product catalog with detailed specifications
- Parts inventory management
- Interactive tables with sorting and pagination
- Dark mode support
- Mobile-friendly interface

## 💻 Tech Stack

- **Frontend**
  - React.js
  - Tailwind CSS
  - Framer Motion
  - React Router DOM
  - TanStack Table (React Table)

- **UI Components**
  - Custom animations
  - Responsive tables
  - Glassmorphism effects
  - Gradient backgrounds

## 🛠️ Getting Started

1. **Clone the repository**
```bash
git clone [repository-url]
cd sk-enterprise
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```
